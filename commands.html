/Join Usage:

/join [team]

About:

This is how you join the game

Other info:

You can only specify a team if you have donated or are the maker of the map currently playing.

/Toggle Usage:

/toggle [setting]

About:

This changes settings specific to you.

Blood:

See blood when a player gets hurt

Death Messages:

Death messages displayed to you

Join Messages:

Join messages displayed to you

Observers:

See other observers while spectating

Picker:

Open a helpful GUI for picking classes and teams

Punishment Messages:

Punishment messsages displayed to you

Scoreboard:

See the scoreboard with game info

Timer:

Show remaining time at the top of the screen

/Report Usage:

/report [player] [reason]

About:

Alerts the staff to the player specified with the reason specified

Other info:

Do not report players unnessasarily or misues the command!

/g Usage:

/g [message]

About:

By default you will speak to only your team. Use this to speak to all players on the server

Other info:

This does not apply in the Lobby server

/msg Usage:

/msg [name] [message]

About:

Sends a message to a certain player. Only that player can see the message

Other info:

Use /r if you recive a msg to quickly reply back

/fr or /friends Usage:

/fr

About:

Sends you a list of your friends and where they are/were on the servers

Other info:

To add friends, go to their stats and click add friend

/match Usage:

/match

About:

Sends you info about match time, number of players, ect

Other info:

N/A

/map Usage:

/map

About:

Sends you info about map rules, authors, max players ect

Other info:

Make sure you read the map rules before playing any map you don't know!

/rot Usage:

/rot

About:

Sends you info about the rotation like next map.

Other info:

To go to the second page to /rot 2
