Welcome

Overcast Network is a Minecraft PvP server started in December by MonsieurApple and Anxuiz. The server runs on the Bukkit API and is powered by many custom plugins, mainly PGM. Overcast also has a website where you can check your stats, socialize with your friends on the forums or just check the revisions for the latest changes to plugins or the website. You can also go to this blog to keep up with the latest news about Overcast Network
